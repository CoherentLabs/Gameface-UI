The `RadioGroup` component lets you render a set of mutually-exclusive options in your UI, where only one choice can be selected at a time.
Under the hood it uses Solid's context API and it works in conjunction with the `RadioButton` component.

## Usage

To create a radio options in your UI, you need to wrap the `RadioButton` components with the `RadioGroup` component. 

```tsx
import RadioGroup from '@components/Basic/RadioGroup/RadioGroup';
import RadioButton from '@components/Basic/RadioButton/RadioButton';

const App = () => {
    return (
        <>
            <RadioGroup>
                <RadioButton selected value="red">red</RadioButton>
                <RadioButton value="green">green</RadioButton>
                <RadioButton value="blue">blue</RadioButton>
            </RadioGroup>
        </>
    );
};

export default App;
```

## API

### Props

| Prop Name  | Type                                   | Default     | Description                                                                                                                                                                                  |
| ---------- | -------------------------------------- | ----------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `style`    | `JSX.CSSProperties`                    | `{}`        | Inline styles to apply directly to the component's root element.                                                                                                                             |
| `class`    | `string`                               | `""`        | Additional CSS classes to apply to the component                                                                                                                                             |
| `ref`      | `RadioGroupRef \| undefined`           | `undefined` | A reference to the component that gives you access to its methods and the underlying HTML element. Useful if you need to set the selected option programmatically.                                                             |
| `disabled` | `boolean`                              | `false`     | Specify if the radio group is disabled                                                                                                                                                            |
| `onChange` | `(selected: string) => void            | `undefined` | A function that is called every time the selected option changes. It can be used to retrieve the selected option's value. |

### Methods

|Method |Parameters |Return Value |Description |
|---|---|---|---|
| `changeOption ` | `newOption: string`  | `void` | Sets new selected option. |

## Guide

### Retrieve the value of the selected option when it changes

To retrieve the value of the selected option:

- Use the `onChange` prop to respond when the selected option changes

#### Steps 

1. Provide a callback to the `onChange` prop to respond to state changes.
2. Inside the callback, access the selected option's value via the `selected` argument.

```tsx
import RadioGroup from '@components/Basic/RadioGroup/RadioGroup';
import RadioButton from '@components/Basic/RadioButton/RadioButton';

const App = () => {
    const getValue = (selected: string) => {
        console.log(selected);
    }
    
    return (
        <>
            <RadioGroup onChange={getValue}>
                <RadioButton selected value="red">red</RadioButton>
                <RadioButton value="green">green</RadioButton>
                <RadioButton value="blue">blue</RadioButton>
            </RadioGroup>
        </>
    );
};

export default App;
```

### Set the selected option Programmatically 

The `RadioGroup` component lets you set the selected option through code.

#### Steps

1. Declare a `ref` variable with type `RadioGroupRef` and pass it to the `RadioGroup` component.
2. Call `ref.changeOption()` from anywhere (e.g., a keyboard event or external button) to set a new selected option programmatically.

```tsx
import RadioGroup, { RadioGroupRef } from '@components/Basic/RadioGroup/RadioGroup';
import RadioButton from '@components/Basic/RadioButton/RadioButton';

const App = () => {
    let RadioGroupRef!: RadioGroupRef;

    const handleKeyPress = (e: KeyboardEvent) => {
        if (e.keyCode !== 13) return
        
        RadioGroupRef.changeOption('green');
    }
    
    return (
        <>
            <RadioGroup ref={RadioGroupRef} keypress={handleKeyPress}>
                <RadioButton selected value="red">red</RadioButton>
                <RadioButton value="green">green</RadioButton>
                <RadioButton value="blue">blue</RadioButton>
            </RadioGroup>
        </>
    );
};

export default App;
```
