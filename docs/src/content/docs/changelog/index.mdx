## 1.0.10 :badge[Latest]{variant="note"}

<Badge text="New" variant="tip" size="small" /> Add a guide on [Working with state](/concepts/working-with-state).

<Badge text="Feature" variant="success" size="small" /> 
[Image](/components/media/image#fill-the-container), [LiveView](/components/media/live-view#fill-the-container), 
[BackgroundImage](/components/media/background-image#props), and 
[MaskImage](/components/media/mask-image#props): added `fill` prop to all media components.

<Badge text="Feature" variant="success" size="small" /> 
[Top](/components/layout/top#props), 
[Content](/components/layout/content#props) and 
[Bottom](/components/layout/bottom#props): added `basis` prop to set the `flex-basis` as a percentage. Allowing for quick layout customization.

<Badge text="Feature" variant="success" size="small" /> [ColorPicker](/components/complex/color-picker#props): added `size` prop for quick size customization. 
Available sizes: `S`, `M`, `L`, and `XL`.

<Badge text="Bug" variant="danger" size="small" /> [Dropdown](/components/basic/dropdown): fixed dropdown trigger displaying the value of an option and not the content related to it.

<Badge text="Bug" variant="danger" size="small" /> [Dropdown](/components/basic/dropdown): fixed dropdown options causing overflow when not visible.

<Badge text="Change" variant="note" size="small" /> [Dropdown](/components/basic/dropdown#props): make dropdown options invert their position when not enough space. Exposed `inverted-class` prop to customize the inverted state.

<Badge text="Bug" variant="danger" size="small" /> [Dropdown](/components/basic/dropdown): fixed `onChange` not triggering when the changed option's value is `""`.

<Badge text="Feature" variant="success" size="small" /> [Scroll](/components/layout/scrol#scrollcontent-required): Expose the inner html element of the scroll's content with a `ref`.

<Badge text="New" variant="tip" size="small" /> Core [components](/components) for building intuitive game user interfaces.

<Badge text="Feature" variant="success" size="small" /> Added a [Vite plugin](/concepts/plugins#vite-solid-style-to-css-plugin) to convert `style` properties to `class`, enhancing performance by reducing inline styles and leveraging CSS for better caching and reusability.

<Badge text="Feature" variant="success" size="small" /> Simplified importing of UI assets with a new [mechanism](/concepts/assets).
